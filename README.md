Siber Dünyasına Yeniden Giriş adına ip scanner tool ( ihtiyaç var gibi duruyordu :D )


> git clone https://github.com/lum1nat3sec/ip-scanner.git


> cd ip-scanner.git


> pip install -r requirements.txt


> python ipscan.py
